# 🚨 FraudGuard – Government Fraud Detection System

FraudGuard is a full-stack, real-time fraud detection system designed to identify anomalies in government welfare schemes. It uses a combination of rule-based logic and machine learning (Isolation Forest) to detect suspicious activities such as duplicate beneficiaries, abnormal claims, and fraud clusters.

## 🚀 Features
- Detects fraud using rule-based and ML techniques  
- Identifies duplicate identities and abnormal transactions  
- Visualizes fraud clusters using network graphs  
- Supports CSV data upload and real-time processing  
- Sends automated alerts for critical fraud cases  
- Interactive dashboard for monitoring and analysis  

## 🛠️ Tech Stack
- Frontend: React, TypeScript, Tailwind CSS  
- Backend: Python, Pandas, Scikit-learn  
- Tools: Streamlit, GitHub, Vercel, EmailJS  

## ⚙️ How It Works
1. Upload beneficiary dataset (CSV)  
2. System processes and validates data  
3. Rule-based checks flag basic fraud patterns  
4. ML model detects anomalies  
5. Dashboard displays results and alerts
